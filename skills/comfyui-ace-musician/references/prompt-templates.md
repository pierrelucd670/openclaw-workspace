## Tips avancés

### Structure explicite
```
"intro verse chorus bridge outro, [genre], [instruments], [mood]"
```

### Prompt stacking
```
"epic orchestral, strings brass choir, 120bpm, D minor, 
 cinematic crescendo, 4 min intro-verse-chorus"
```

### Metadata override
```
"[genre], bpm:120, key:C#minor, time_sig:6/8, 
 structure:A-B-A-B-C, duration:300s"
```

### Référence audio (cover/repaint)
```
Ajouter "cover of [reference_audio.wav]" ou 
"repaint [start_time-end_time] of [audio.wav]"
```
